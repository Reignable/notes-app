# Take Note! (Notes app)

This is a React application for creating, editing, and deleting notes.
Notes are stored in the `db.json` file and served up by [Json-server](https://github.com/typicode/json-server).

The app runs on port 3000 and the server on 3001

Note pages have their own URL so they can be bookmarked. Their titles can be edited by clicking on the title in the note view.

### Run

1. Install dependencies
   - `yarn`
2. Run server and web app
   - `yarn start`

### Test

1. Install dependencies
   - `yarn`
2. Run the tests
   - `yarn test`
